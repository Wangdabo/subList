## `widget` 自定义属性

参数 | 说明 | 类型 | 默认值
----|------|-----|------
config | 前端配置项说明，[见官网](http://fex.baidu.com/ueditor/#start-config) | `Object` | -
loadingTip | 初始化提示文本 | `string` | `加载中...`
onContentChange | 编辑器内容发生改变时会触发该事件 | `Function` | -
